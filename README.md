## Getting Started

### Powershell

1. Open VSCode as Administrator
2. Open a Powershell terminal session
3. Run `Set-ExecutionPolicy Unrestricted`
4. Run `corepack enable`
5. Run `yarn`
6. Run `yarn start:dev`
